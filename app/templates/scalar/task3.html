Overview

In Task 3, you will identify regions in South America with relatively high amounts of snow cover. You will need to navigate to the tiles representing South America in the data set, and find tiles in this area that meet the goal requirements described below.

About the Data Set : The Task 3 data set is the same data set used for Task 1. This data set is is a 2D array containing Normalized Difference Snow Index (NDSI) calculations from the MODIS satellite imagery data set. The two dimensions are latitude (x) and longitude (y). A high NDSI value (1.0) means that particular latitude/longitude area contains a high amount of snow. A low value (-1.0) means that particular part of the earth does not contain snow. Large bodies of water (i.e. oceans, lakes, rivers) appear as if they contain high amounts of snow. Add a land/sea mask filter to ignore them.

Goal(s) : Find tiles in South America at zoom level X that contain NDSI values greater than .25. Hint: Apply a filter over 'avg_ndsi' values to ignore low NDSI values.
