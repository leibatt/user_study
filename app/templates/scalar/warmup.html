What You Need to Know

About the Data Set: The warmup data set is a 2D array containing Normalized Difference Vegetation Index (NDVI) calculations from the MODIS data set. The two dimensions are latitude (x) and longitude (y). A high NDVI value (1.0) means that particular latitude/longitude area contains a high amount of vegetation (trees, shrubs, grass etc.). A low value (-1.0) means that particular part of the earth does not contain vegetation.
